 Given IP = 210.15.131.141 and subnet mask = 255.255.255.224. Calculate the last host of the last subnet??

210.15.131.202

 210.15.131.162

  210.15.131.254

Correct answer

 none of these 

Solution

Hide

A given IP is class C, so the first three octets part will be fixed.

 

The last octet bit binary representation of the subnet mask is 255.255.255.11100000 (3 bits are subnet and 5 bits are for the host)

Therefore, possible subnet bits are 2^3 = (000(first),001,010,011,100,101,110,111(last)) 

 

The last host of the last subnet is 210.15.131.111111110(will all 1s except last bit)

i.e. 210.15.131.254

so correct answer would be 210.15.131.254

Question 16

You didn't Attempt

Your time taken: 0s
Network Criteria

Suppose a packet of length 1800 bytes with minimum header size enter a local area network with MTU (Maximum transfer unit ) of 500 bytes, then how many fragments will be generated from the packet and what is the size of the last packet??

3, 300

4, 360

Correct answer

5, 320

 3, 340

Solution

Hide

Length of the packet = 1800 bytes

Therefore, Total data bytes = 1800 - 20 =1780 (Payload)

Given MTU = 500

Data bytes in first fragment = 500 - 20 = 480

Data bytes in second fragment = 500 - 20 = 480

Data bytes in third fragment = 500 - 20 = 480,

And data bytes in last fragment = 340

Hence size of the last packet= 340 + 20 = 360

Therefore the size of the last packet is 360 bytes. And the total no of fragmented packets is 4.

 Computer A has 40 MB to send on a network and transmits the data in burst at 8 Mbps. The maximum transmission rate across routers in the network is 6 Mbps. If computer A’s transmission is shaped using a leaky bucket. The capacity that the queue in the bucket must hold so that no data is discarded is ______(in MB).


10

Correct answer

Solution

Hide

Total data =40*8 Mb (convert the byte to bit)

Time for computer to transmit data = 40*8 Mb / 8Mbps = 40 sec

Maximum transmission rate = 6Mbps

Actual data sent on network in 40 sec = 6Mbps*40 = 240Mb = 30MB (change into byte)

Bucket must hold = 40MB-30 MB=10MB
