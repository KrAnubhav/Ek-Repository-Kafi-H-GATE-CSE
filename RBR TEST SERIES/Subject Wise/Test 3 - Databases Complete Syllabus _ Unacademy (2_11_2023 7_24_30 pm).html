Third Normal Form

The best normal form of relation schema R(A,B,C,D) along with the set of functional dependencies F={ABC,ABD,CA,DB} is

BCNF

3NF

Correct answer

2NF

1NF

Solution

Hide

Given relation R(A,B,C,D)

FDs: ABC

          ABD

             CA

            DB

(AB)^+=ABCD

(BC)^+=BCAD

(AD)^+=ADBC

(CD)^+=CDAB


  • Candidate keys are {AB, BC, AD, CD}. So, all attributes are prime attributes.
  • So, the relational schema is in 3NF.
  • C and D are not super keys, so CA and DB does not satisfy the condition of BCNF.

Types Functional Dependency

Consider the following set of functional dependencies on schema (A, B, C):

A->BC

B->C

A->B

AB->C

What is the canonical cover for F?

AC, BC

AB, BC

Correct answer

BC, AB, ABC

AB ,CB

Solution

Hide

Given FD set:

ABC, BC, AB, ABC

Step :

AB,AC,BC,AB,ABC

  • (A)^+=AB , without A->B , B is derived, so we can remove A->B .
  • (A)+=ABC , without A->C, C is derived , so we can remove A->C.
  • B->C and A->B cannot derive C and B respectively without them, so we cannot remove them.
  • Without AB->C also, we can derive C from B->C. So AB->C can be removed.
  • Hence Answer is (b).

I. 2-phase locking

II. Time-stamp ordering


I only

Correct answer

II only

Both I and II

Neither I nor II

Solution

Hide
